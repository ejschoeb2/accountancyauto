  <name>Task 1: Create lib/storage/google-drive.ts — GoogleDriveProvider class</name>
  <files>lib/storage/google-drive.ts</files>
  <action>
    Create `lib/storage/google-drive.ts`. This file implements the `StorageProvider` interface for Google Drive.

    **Constructor:**
    ```typescript
    export class GoogleDriveProvider implements StorageProvider {
      constructor(private readonly orgId: string, private readonly rootFolderId: string) {}
    ```
    The `orgId` is used to look up token columns from the DB inside each method. The `rootFolderId` is the Drive file ID of the `Prompt/` root folder (from `organisations.google_drive_folder_id`).

    **Private helper — `getOrgCredentials()`:**
    Fetches `google_access_token_enc`, `google_refresh_token_enc`, `google_token_expires_at` from organisations via admin client. Throws `'Google Drive not connected'` if `google_access_token_enc` is null.

    **Private helper — `findOrCreateFolder(drive, parentId, name): Promise<string>`:**
    Searches for an existing folder by name under parentId using Drive API list query:
    ```
    q: `name='${escapedName}' and '${parentId}' in parents and mimeType='application/vnd.google-apps.folder' and trashed=false`
    ```
    Escape single quotes in folder name: `name.replace(/'/g, "\\'")`.
    Returns first result ID if found. If not found, creates the folder with `files.create({requestBody: {name, mimeType: 'application/vnd.google-apps.folder', parents: [parentId]}, fields: 'id'})` and returns the new folder ID.
    This prevents the duplicate folder problem documented in Research pitfall #3.

    **`upload(params: UploadParams): Promise<{ storagePath: string }>`:**
    1. Call `getOrgCredentials()` to get token columns
    2. Build `GoogleCredentials` object and call `withTokenRefresh(creds, async (auth) => { ... })`
    3. Inside the callback: instantiate `google.drive({ version: 'v3', auth })`
    4. Create folder hierarchy lazily using `findOrCreateFolder` for each level:
       - Level 1: `params.clientName` under `this.rootFolderId`
       - Level 2: `params.filingTypeId` under the client folder ID
       - Level 3: `params.taxYear` under the filing type folder ID
    5. Convert `params.file` (Buffer/Uint8Array) to a Node.js `Readable` stream: `Readable.from(Buffer.from(params.file))`
    6. Call `drive.files.create({requestBody: {name: params.originalFilename, parents: [leafFolderId]}, media: {mimeType: params.mimeType, body: readable}, fields: 'id'})`
    7. Return `{ storagePath: file.data.id! }` — the Drive file ID IS the storagePath for Google Drive documents

    **`getBytes(storagePath: string): Promise<Buffer>`:**
    1. Call `getOrgCredentials()` and `withTokenRefresh(creds, async (auth) => { ... })`
    2. `drive.files.get({ fileId: storagePath, alt: 'media' }, { responseType: 'stream' })`
    3. Collect the stream into a Buffer using the `streamToBuffer` helper:
       ```typescript
       async function streamToBuffer(stream: NodeJS.ReadableStream): Promise<Buffer> {
         return new Promise((resolve, reject) => {
           const chunks: Buffer[] = [];
           stream.on('data', (chunk) => chunks.push(Buffer.from(chunk)));
           stream.on('error', reject);
           stream.on('end', () => resolve(Buffer.concat(chunks)));
         });
       }
       ```

    **`getDownloadUrl(_storagePath: string): Promise<{ url: string }>`:**
    MUST throw — drive.file scope cannot produce public URLs. All download routes must use getBytes() instead for Google Drive:
    ```typescript
    async getDownloadUrl(_storagePath: string): Promise<{ url: string }> {
      throw new Error(
        'getDownloadUrl() is not available for Google Drive (drive.file scope). ' +
        'Use getBytes() via the server-proxy download route instead.'
      );
    }
    ```

    **`delete(storagePath: string): Promise<void>`:**
    1. `getOrgCredentials()` + `withTokenRefresh(creds, async (auth) => { ... })`
    2. `drive.files.delete({ fileId: storagePath })`

    Imports:

    ```typescript
    import { withTokenRefresh, type GoogleCredentials } from './token-refresh';

    ```

    ```typescript
        case 'google_drive':
          if (!orgConfig.google_drive_folder_id) {
            throw new Error('Google Drive is connected but google_drive_folder_id is missing from org config');
    ```
