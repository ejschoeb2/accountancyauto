Add the wizard gate AFTER the `readOnly = await isOrgReadOnly(orgId)` line:

```typescript
    // Gate: members must complete the wizard before accessing the dashboard
    if (role === "member") {
      const { getMemberSetupComplete } = await import("@/app/actions/settings");
      const setupComplete = await getMemberSetupComplete();
      if (!setupComplete) {
        redirect("/setup/wizard");
      }
    }
```

Use a dynamic import (`await import(...)`) for getMemberSetupComplete to avoid a top-level import that would affect all layout renders. This is safe — the import is cached after the first call.

- ONLY check getMemberSetupComplete when `role === "member"`. Admins went through onboarding (not the wizard) and must NEVER be redirected to /setup/wizard.
- This check goes INSIDE the existing try/catch. If getOrgContext() throws, the check is skipped — the existing catch block handles that gracefully.
- Do NOT import getMemberSetupComplete at the top of the file — dynamic import keeps this conditional.

Add `redirect` import from "next/navigation" to the existing imports at the top if not already present.
