**Create `lib/auth/org-context.ts`** — a helper that extracts org_id from the current user's JWT claims. This will be used by server actions that need to explicitly pass org_id (e.g., app_settings queries, inserts).

```typescript
import { createClient } from '@/lib/supabase/server';

export async function getOrgId(): Promise<string> {
  const supabase = await createClient();
  const { data: { user }, error } = await supabase.auth.getUser();

  if (error || !user) {
    throw new Error('Not authenticated');
  }

  const orgId = user.app_metadata?.org_id;
  if (!orgId) {
    throw new Error('No org_id in user session — user may not be assigned to an organisation');
  }

  return orgId;
}

export async function getOrgContext(): Promise<{ orgId: string; orgRole: string }> {
  const supabase = await createClient();
  const { data: { user }, error } = await supabase.auth.getUser();

  if (error || !user) {
    throw new Error('Not authenticated');
  }

  const orgId = user.app_metadata?.org_id;
  const orgRole = user.app_metadata?.org_role;

  if (!orgId) {
    throw new Error('No org_id in user session');
  }

  return { orgId, orgRole: orgRole || 'member' };
}
```

Pattern for all settings reads:
```typescript
export async function getSendHour(): Promise<number> {
  const supabase = await createClient();
  const orgId = await getOrgId();
  const { data } = await supabase
    .from("app_settings")
    .select("value")
    .eq("org_id", orgId)
    .eq("key", "reminder_send_hour")
    .single();
  return data ? parseInt(data.value, 10) : 9;
}
```

Pattern for all settings writes:
```typescript
// Upsert now needs both org_id and key
const { error } = await supabase
  .from("app_settings")
  .upsert(
    { org_id: orgId, key: "reminder_send_hour", value: String(hour) },
    { onConflict: "org_id,key" }
  );
```
