**lib/documents/ocr.ts — extractPdfText() and extractFieldsForType()**

`extractPdfText(buffer: Buffer): Promise<OcrResult>`
- Calls `pdfParse(buffer)` from `pdf-parse-debugging-disabled`
- If pdfParse throws → re-throw (caller catches as corrupt PDF signal)
- Returns `{ text: string, numpages: number, isImageOnly: boolean }`
- `isImageOnly` is true when `text.trim().length < 50 * Math.max(numpages, 1)`

`extractFieldsForType(code: string, rawText: string): { taxYear: string | null, employer: string | null, payeRef: string | null }`
- Normalises text first: `rawText.replace(/\s+/g, ' ')` for multi-line label matching
- Applies per-type patterns (case-insensitive):

P60:
  - taxYear: `/Tax year to 5 April (\d{4})/i` → capture group 1
  - employer: `/Employer'?s? name[:\s]*([A-Za-z0-9 &.,'-]{2,80})/i` → capture group 1, trimmed
  - payeRef: anchor to label first — `/(?:PAYE reference|Employer'?s? PAYE ref)[:\s]*(\d{3}\/[A-Z0-9]{1,10})/i`

P45:
  - taxYear: `/(?:Year|year) to 5 April (\d{4})/i`
  - employer: `/Employer'?s? name[:\s]*([A-Za-z0-9 &.,'-]{2,80})/i`
  - payeRef: `/(?:Employer PAYE reference|PAYE reference)[:\s]*(\d{3}\/[A-Z0-9]{1,10})/i`

SA302:
  - taxYear: `/Year ended 5 April (\d{4})/i`
  - employer: null (SA302 is individual — no employer field)
  - payeRef: null (UTR-based, not PAYE)

P11D:
  - taxYear: `/(?:for the )?[Yy]ear ended 5 April (\d{4})/i`
  - employer: `/Employer'?s? name[:\s]*([A-Za-z0-9 &.,'-]{2,80})/i`
  - payeRef: `/(?:PAYE reference)[:\s]*(\d{3}\/[A-Z0-9]{1,10})/i`

Unknown code: return `{ taxYear: null, employer: null, payeRef: null }`

**lib/documents/integrity.ts — runIntegrityChecks()**

`runIntegrityChecks(buffer: Buffer, mimeType: string, clientId: string, supabase: SupabaseClient): Promise<IntegrityResult>`

```typescript
interface IntegrityResult {
  passed: boolean;
  fileSizeBytes: number;
  pageCount: number | null;      // null for non-PDFs
  isDuplicate: boolean;
  sha256Hash: string;
  rejectionReason: string | null;
}
```

Checks in order:
1. **File size**: if buffer.length > 20 * 1024 * 1024 (20MB) → passed=false, rejectionReason='File exceeds 20MB limit'
2. **Duplicate hash**: compute SHA-256 via `crypto.createHash('sha256').update(buffer).digest('hex')`. Query `client_documents` for existing row with same client_id AND file_hash. If found → isDuplicate=true, passed=false, rejectionReason='This file has already been uploaded'
3. **Page count** (PDF only): if mimeType==='application/pdf', call extractPdfText to get numpages. If numpages > 50 → passed=false, rejectionReason='Document exceeds 50 page limit'. Store numpages in result.
4. Non-PDF files: pageCount=null (skip page count check)

Always populate sha256Hash and fileSizeBytes regardless of pass/fail.
If all checks pass: passed=true, rejectionReason=null.

Note: runIntegrityChecks does NOT call pdfParse for corrupt PDFs — if pdf-parse throws during page count check, catch and set pageCount=null (do not reject for this reason; corrupt detection happens in classifyDocument).

**lib/documents/classify.ts — extended ClassificationResult and classifyDocument()**

Extend `ClassificationResult` interface by adding Phase 21 fields:
```typescript
export interface ClassificationResult {
  documentTypeId: string | null;
  documentTypeCode: string | null;
  filingTypeId: string | null;
  confidence: 'high' | 'medium' | 'low' | 'unclassified';
  // Phase 21 additions (with backward-compatible defaults):
  extractedTaxYear: string | null;
  extractedEmployer: string | null;
  extractedPayeRef: string | null;
  extractionSource: 'ocr' | 'keyword' | 'rules';
  isCorruptPdf: boolean;
  isImageOnly: boolean;
}
```

Change `classifyDocument` signature to accept optional buffer:
```typescript
export async function classifyDocument(
  filename: string,
  mimeType: string,
  supabase: SupabaseClient,
  buffer?: Buffer
): Promise<ClassificationResult>
```

Internal logic:
1. Existing keyword match runs first (unchanged).
2. After keyword match, check if: `buffer !== undefined && mimeType === 'application/pdf' && HMRC_OCR_TYPES.has(entry.code)`
   - `HMRC_OCR_TYPES = new Set(['P60', 'P45', 'SA302', 'P11D'])`
3. If in OCR path:
   - try: `const ocr = await extractPdfText(buffer)`
     - If `ocr.isImageOnly` → return result with confidence='unclassified', all extracted fields null, extractionSource='rules', isImageOnly=true, isCorruptPdf=false
     - Otherwise: call `extractFieldsForType(entry.code, ocr.text)`, return with extractionSource='ocr', isCorruptPdf=false, isImageOnly=false, confidence = extracted.taxYear ? 'high' : 'medium'
   - catch: → return with isCorruptPdf=true, all null fields, extractionSource='rules', confidence='unclassified'
4. For non-OCR paths (no buffer, or non-HMRC type, or non-PDF): return with extractionSource='keyword', all extracted fields null, isCorruptPdf=false, isImageOnly=false.
5. For the generic PDF fallback (no keyword match, mimeType=application/pdf): extractionSource='keyword', confidence='low'.
6. For unclassified: extractionSource='rules', confidence='unclassified'.

IMPORTANT: All existing callers of `classifyDocument(name, mimeType, supabase)` WITHOUT a buffer must continue to work without changes. The buffer parameter is optional.
