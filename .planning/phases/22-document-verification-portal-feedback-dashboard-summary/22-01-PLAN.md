  <action>
Create a new Supabase migration file. The Phase 21 migration (20260225000001_phase21_ocr_columns.sql) added extraction_source with an inline CHECK constraint. PostgreSQL auto-names inline CHECK constraints as {table}_{column}_check. The constraint must be dropped and recreated with the new value.

Migration content:
```sql
-- Phase 22: Add 'manual' to extraction_source CHECK constraint
-- Allows accountants to manually correct OCR-extracted fields in the dashboard.
-- The inline CHECK added in Phase 21 must be dropped and replaced — PostgreSQL does not
-- support in-place modification of CHECK constraints.
--
-- Auto-generated constraint name from Phase 21 ADD COLUMN inline CHECK:
--   client_documents_extraction_source_check
-- Using IF EXISTS defensively in case the name differs on a clean remote.

ALTER TABLE client_documents
  DROP CONSTRAINT IF EXISTS client_documents_extraction_source_check;

ALTER TABLE client_documents
  ADD CONSTRAINT client_documents_extraction_source_check
  CHECK (extraction_source IN ('ocr', 'keyword', 'rules', 'manual'));
```

After creating the file, apply the migration to the remote database:
```bash
npx supabase db push
```


**Part A — lib/documents/integrity.ts:**

Add an optional `options` parameter to `runIntegrityChecks` to support bypassing the duplicate check when the client has confirmed they want to proceed with a known duplicate. Keep the signature backward-compatible (optional param, existing callers unchanged).

Change the function signature from:
```typescript
export async function runIntegrityChecks(
  buffer: Buffer,
  mimeType: string,
  clientId: string,
  supabase: SupabaseClient
): Promise<IntegrityResult>
```

To:
```typescript
export async function runIntegrityChecks(
  buffer: Buffer,
  mimeType: string,
  clientId: string,
  supabase: SupabaseClient,
  options?: { skipDuplicate?: boolean }
): Promise<IntegrityResult>
```

Inside the function, in the duplicate hash check section (Check 2), wrap the duplicate detection logic in an `if (!options?.skipDuplicate)` guard:
```typescript
// ── Check 2: Duplicate hash ───────────────────────────────────────────────
if (!options?.skipDuplicate) {
  const { data: duplicates, error: dupError } = await supabase
    .from('client_documents')
    .select('id')
    .eq('client_id', clientId)
    .eq('file_hash', sha256Hash)
    .limit(1);

  if (!dupError && duplicates && duplicates.length > 0) {
    return {
      passed: false,
      fileSizeBytes,
      pageCount: null,
      isDuplicate: true,
      sha256Hash,
      rejectionReason: 'This file has already been uploaded',
    };
  }
}
```
