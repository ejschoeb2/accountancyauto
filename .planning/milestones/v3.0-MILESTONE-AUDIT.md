**Low-severity integration gaps (intentional design decisions):**

2. Invite accept flow loses `?token=` through auth callback for unauthenticated users (D-13-02-03) — must re-click invite link

### Operational Setup Required (Not Code Gaps)

These are manual configuration steps needed before the platform is live:
1. Enable Custom Access Token Hook in Supabase Dashboard > Authentication > Hooks
2. Set `is_super_admin = true` in app_metadata for operator accounts
3. Configure Stripe webhook endpoint in Stripe Dashboard
4. Populate all `STRIPE_PRICE_*` env vars with real Stripe Price IDs
5. Set `STRIPE_WEBHOOK_SECRET` env var
6. Ensure founding org has `postmark_server_token` populated
